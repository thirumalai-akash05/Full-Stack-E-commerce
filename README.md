# FreshBite - Online Food Ordering

A frontend-only online food ordering platform built for the Free Web Development Internship offered by Data Alcott Systems.

## About

FreshBite lets users browse restaurants, view menus, add items to a cart, apply coupons, check out, and track simulated orders. It is a pure frontend project built with HTML, CSS, and JavaScript, using in-memory JS arrays/objects for data instead of a database.

## Features

- Home page with categories, featured dishes, and offers
- Restaurant listing with filters (cuisine, rating, price)
- Menu pages with dietary filters (Veg, Non-Veg, Vegan, Gluten-Free)
- Product detail pages with ingredients and nutrition info
- Shopping cart with quantity adjustment and price calculation
- Checkout with address, payment method, and delivery time selection
- Simulated login/register and user account with order history
- Order tracking simulation
- Customer reviews and ratings
- Coupon codes
- Wishlist

## Tech Stack

- HTML5
- CSS3
- JavaScript
